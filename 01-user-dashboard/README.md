# User Dashboard (React)

A simple React application that displays a list of users and their
individual details using API integration and React Router.

This project was built to understand real-world React concepts like
routing, dynamic pages, API fetching, and state management.

## Features

- Home page
- Users list page (data fetched from API)
- User details page using dynamic routing
- Navigation using React Router
- Loading and error handling
- Clean and modular component structure

## Tech Stack

- React
- React Router DOM
- JavaScript (ES6+)
- HTML & CSS
- Vite (build tool)

## API Used

- JSONPlaceholder Users API  
  https://jsonplaceholder.typicode.com/users
